# 🚢 PRODIGY_DS_02

## 📌 Task 2: Titanic Dataset - Data Cleaning & Exploratory Data Analysis (EDA)

This project focuses on performing **data cleaning and exploratory data analysis (EDA)** on the Titanic dataset to uncover patterns, trends, and relationships between variables.

---

## 📂 Dataset

* Titanic dataset (Kaggle)
* Contains passenger details such as:

  * Age
  * Gender
  * Class
  * Fare
  * Survival status

---

## 🧹 Data Cleaning Steps

* Handled missing values:

  * Filled `Age` with median
  * Filled `Embarked` with mode
* Dropped `Cabin` column (too many missing values)
* Verified dataset integrity

---

## 📊 Exploratory Data Analysis (EDA)

* Age distribution analysis
* Fare distribution visualization
* Survival analysis based on:

  * Gender
  * Passenger class

---

## 📈 Key Insights

* Females had higher survival rates than males
* Passengers in higher class had better survival chances
* Fare distribution is highly skewed
